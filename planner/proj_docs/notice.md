Для напоминаний было решено добавить 5 новых фишек:
1. Повторяющиеся напоминания - раз в n время (дни\недели\месяцы...)
2. Через бота кнопку "отложить напоминание" и на сколько отложить это напоминание
3. Напоминание для задач с дедлайном - пока таковых нет, только у целей есть дедлайн
4. В плане на день для задачи можно сделать напоминание - какой-нибудь checkbox "напомнить" и бот в нужное
время пришлет напоминание о том, что есть вот такая задача
5. Ответы на напоминания - сделано\прочитано\удалить (или не удалить, придумаю потом), если
не сделано, то через какое-то время бот пришлёт еще одно напоминание и.т.д
6. Работа умного дома с напоминаниями
