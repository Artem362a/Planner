ПОЛЬЗОВАТЕЛЬСКОЕ СОГЛАШЕНИЕ

1. Общие положения
1.1. Настоящее Пользовательское соглашение регулирует порядок использования сервиса планирования задач, целей и обратной связи (далее — Сервис).
1.2. Используя Сервис и регистрируя учетную запись, пользователь подтверждает, что ознакомился с условиями настоящего Соглашения и принимает их в полном объеме.
1.3. Если пользователь не согласен с условиями Соглашения, он обязан прекратить использование Сервиса.

2. Термины
2.1. Сервис — веб-приложение для ведения планов на день, неделю, целей, шаблонов и обращений через форму обратной связи.
2.2. Пользователь — физическое лицо, использующее Сервис.
2.3. Оператор — владелец Сервиса, организующий его работу и обработку данных пользователей.

3. Регистрация и аккаунт
3.1. Для использования отдельных функций Сервиса пользователь создает учетную запись.
3.2. Пользователь обязуется предоставить достоверные данные при регистрации.
3.3. Пользователь самостоятельно обеспечивает сохранность данных для входа в учетную запись.
3.4. Все действия, совершенные с использованием учетной записи пользователя, считаются совершенными самим пользователем до момента уведомления Оператора о компрометации доступа.

4. Функциональность Сервиса
4.1. Пользователю могут быть доступны:
- ведение плана на день;
- ведение плана на неделю;
- создание шаблонов;
- ведение целей и этапов;
- направление обращений через форму обратной связи.
4.2. Оператор вправе изменять набор функций Сервиса, интерфейс и порядок его работы без предварительного уведомления пользователя.

5. Права и обязанности пользователя
5.1. Пользователь обязуется:
- не нарушать законодательство Российской Федерации;
- не использовать Сервис для противоправных целей;
- не размещать вредоносный код, спам, оскорбительные или незаконные материалы;
- не предпринимать действий, направленных на нарушение работы Сервиса.
5.2. Пользователь вправе использовать Сервис исключительно в личных целях, если иное не согласовано отдельно.

6. Права Оператора
6.1. Оператор вправе:
- временно ограничивать доступ к Сервису для проведения технических работ;
- изменять функциональность Сервиса;
- ограничивать или прекращать доступ пользователя при нарушении Соглашения;
- удалять материалы и данные, размещенные с нарушением закона или настоящего Соглашения.

7. Персональные данные
7.1. Порядок обработки персональных данных определяется Политикой обработки персональных данных.
7.2. Пользователь подтверждает, что ознакомился с Политикой обработки персональных данных до начала использования Сервиса.

8. Ограничение ответственности
8.1. Сервис предоставляется по принципу «как есть».
8.2. Оператор не гарантирует бесперебойную и безошибочную работу Сервиса.
8.3. Оператор не несет ответственности за невозможность использования Сервиса по причинам, не зависящим от Оператора.
8.4. Оператор не несет ответственности за утрату данных пользователя, если такая утрата произошла вследствие действий пользователя, третьих лиц, либо технических сбоев, которые невозможно было разумно предотвратить.

9. Интеллектуальные права
9.1. Исключительные права на Сервис, его дизайн, программный код, структуру и элементы интерфейса принадлежат Оператору либо используются им на законных основаниях.
9.2. Пользователь не вправе копировать, распространять, декомпилировать или иным образом использовать элементы Сервиса вне пределов обычного пользовательского функционала.

10. Прекращение использования
10.1. Пользователь вправе прекратить использование Сервиса в любое время.
10.2. Оператор вправе удалить или заблокировать учетную запись пользователя при нарушении условий Соглашения.

11. Изменение Соглашения
11.1. Оператор вправе изменять настоящее Соглашение.
11.2. Новая редакция вступает в силу с момента ее публикации в Сервисе, если иное не указано в самой редакции.

12. Контакты
12.1. По всем вопросам, связанным с использованием Сервиса, пользователь может обратиться по контактам, указанным в разделе контактов Сервиса.
